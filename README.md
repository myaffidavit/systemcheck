# myAffidavit SystemCheck Application

Please follow these steps on Windows:
Download the attached ZIP.
Right-click the ZIP → Properties → check Unblock → Apply.
Extract the ZIP to a folder (e.g., Desktop).
Run the .exe inside the extracted folder.
If Windows SmartScreen shows a warning, choose More info → Run anyway.
Once you run it, it will report your system compatibility (CPU/RAM/GPU, etc.), whether your machine meets the requirements.
