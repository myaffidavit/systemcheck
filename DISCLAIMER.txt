--------------------------------------------------------------------------------
                              PURPOSE
--------------------------------------------------------------------------------

This application checks whether your Windows computer meets the minimum 
hardware and software requirements to run the myAffidavit desktop application.

It performs the following local checks:
- Operating system version
- CPU specifications  
- Available RAM
- Available disk space
- Graphics capabilities

--------------------------------------------------------------------------------
                         PRIVACY & DATA COLLECTION
--------------------------------------------------------------------------------

THIS APPLICATION:

- Does NOT collect any personal information
- Does NOT transmit any data over the internet
- Does NOT connect to any external servers or services
- Does NOT install any software on your computer
- Does NOT modify any system settings
- Does NOT access files, documents, or personal data
- Runs ENTIRELY OFFLINE on your local machine

All system information is displayed on-screen only and is not stored, logged,
or transmitted anywhere.

--------------------------------------------------------------------------------
                              USAGE
--------------------------------------------------------------------------------

1. Run the application
2. View the compatibility results on screen
3. Optionally copy the report to clipboard for your reference
4. Close the application when finished

You may delete this application at any time by simply removing the file.
No uninstallation is required.
